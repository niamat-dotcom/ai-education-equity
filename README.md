# AI for Education Equity Dashboard
Fairness-aware AI dashboard analyzing gender-based educational inequality using explainable machine learning.
## Overview
This project uses **fairness-aware artificial intelligence**
to analyze and predict gender-based educational inequality
across countries using real-world enrollment data.

## Key Features
- Gender enrollment trend analysis
- AI-powered inequality risk prediction
- Explainable AI with SHAP (global & local)
- Interactive visual dashboards

## Technology Stack
- Python
- Streamlit
- Scikit-learn
- SHAP
- Plotly & Altair

## Social Impact
This project demonstrates how AI can be applied responsibly
to support educational access and equity, especially in
underrepresented regions.
